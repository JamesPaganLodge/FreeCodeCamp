What is Openedge ABL?

Openedge ABL is known as a 4GL language which has supposedly built upon the functionality of 3GL languages. The purpose of the 4GL languages was to open up the power of programming to non-developers to enable them to write reports and update databases. Openedge both a database query language and a programming language.

Other 4GL languages:

  • PowerBuilder
  • Oracle Application Development Framework
  • Clarion

Openedge is currently on version 11.7.4 with version 12 set to be released in Q4 2018. Version 12 will finally bring multi-threading to the language

Hello World

To output the text "Hello World!" copy the following into the Openedge Editor:

message "Hello World!" view-as alert-box.

This should pop up an alert box on the screen.
